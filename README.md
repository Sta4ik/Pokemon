# Pokemon Collector
Просто сайт на React + JS, где можно ловить рандомных покемонов и собирать свою коллекцию.  

## Что тут вообще есть
- Жмёшь кнопку — ловишь случайного покемона
- Все пойманные покемоны сохраняются
- Есть профиль
- Страница с карточками покемонов

## Почему это круто? 
- Потому что покемоны - бессмертная классика
- Потому что можно словить себе 1000000000000 пикачу
